== Description ==

Blogirame.mk Widget додатокот ги прикажува најпопуларните и најнови блог написи од блоговите кои ги агрегира Blogirame.mk.

== Installation ==

1. Прикачете го „BlogirameMK“ директориумот во „/wp-content/plugins“.
2. Активирајте го додатокот.
3. Отидете во менито „Елементи“ и додадете го виџетот на вашиот веб-сајт.
